# Calculadora JS

Calculadora simples criada com HTML, CSS e JavaScript.

O objetivo deste projeto foi praticar lógica de programação, manipulação da tela com JavaScript e organização de arquivos em um projeto web.

## 🔗 Projeto online

Acesse a calculadora publicada:

https://mthscostaz.github.io/calculadora-js/

## 🚀 Tecnologias usadas

- HTML
- CSS
- JavaScript
- GitHub
- GitHub Pages

## 📚 O que aprendi

Neste projeto, pratiquei:

- Estruturação de uma página com HTML
- Estilização com CSS
- Criação de botões interativos
- Uso de funções em JavaScript
- Manipulação do valor exibido na tela
- Operações matemáticas básicas
- Publicação de projeto com GitHub Pages

## ⚙️ Funcionalidades

- Somar
- Subtrair
- Multiplicar
- Dividir
- Apagar último caractere
- Limpar visor
- Calcular resultado
